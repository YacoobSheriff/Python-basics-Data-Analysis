# Python-for-Data-Analysis
Python fundamentals for data analysis with hands-on examples

This repository contains beginner‑level Python examples focused on fundamentals used in data analysis.

## What’s inside
This project includes:
- basics.py – Variables & if/else conditions  
- loops.py – For & while loops  
- functions.py – Functions in Python  
- file_handling.py – Reading and writing files  
- error_handling.py – Using try/except for error control

## What I learned
In this project I practised:
- Python variables and printing  
- Conditional logic (if/else)  
- Looping (for and while loops)  
- Functions
- File handling (read/write)
- Basic error handling with try/except  

## Tools used
- Python 3  
- GitHub for version control

## How to use
1. Clone the repo  
2. Open Python files  
3. Run scripts with Python to see outputs
